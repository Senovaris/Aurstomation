# Aurstomation - Aurs automation.

Bunch of small additions to for myself as a WoW AddOn.

## Examples
- Auto accept invites
- Auto accept role checks
- Set to "current expansion" in auction house.
- Cross in the middle of the character
- Auto sell and repair
- Automatic zoom settings if 1440p or 1080p.
